Confira as principais novidades do Langflow 1.8, a plataforma open-source para construção de AI agents e aplicações com LLMs. Nesta versão, o foco está em produtividade, observabilidade e gestão de modelos, com recursos que aproximam o desenvolvimento de agentes de cenários reais de produção.


{% include youtubePlayer.html id=page.youtubeId  %}

## Principais pontos

Neste vídeo, exploramos em detalhes os novos recursos do Langflow:

Knowledge Bases nativas para integração de dados e construção de aplicações com contexto (RAG)

Flow Tracing para observabilidade completa da execução dos fluxos e debugging de agentes

LLM Models Management para gerenciar múltiplos modelos e provedores de forma centralizada

Workflow API para expor fluxos como APIs e integrar com aplicações externas

Se você trabalha com Generative AI, agentes autônomos, RAG (Retrieval-Augmented Generation) ou desenvolvimento com LLMs, este update do Langflow traz capacidades importantes para levar seus projetos para produção.
