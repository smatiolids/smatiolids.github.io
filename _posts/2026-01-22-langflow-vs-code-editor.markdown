Gostaria de contar com ajuda da IA para criar componentes customizados no Langflow? (English version below...)

Criei uma extensão para **VSCode** que permite editar o código de componentes do Langflow diretamente no IDE — incluindo **VS Code**, **Cursor** e **IBM BOB**.

Na prática, isso possibilita:

- Criar e manter componentes sem sair do editor
- Usar assistentes de código baseados em IA, como **Codex**, **GitHub Copilot** e outros, durante a edição
- Ganhar produtividade e consistência ao evoluir componentes mais complexos

Se você trabalha com componentes customizados no Langflow, teste e compartilhe seus feedbacks no **Git**.

[English]

Would you like AI assistance when creating custom components in Langflow?
I built a VS Code extension that lets you edit Langflow component code directly in your IDE — including **VSCode**, **Cursor**, and **IBM BOB**.

With this approach, you can:

- Build and maintain custom components without leaving your editor
- Leverage AI code assistants like **Codex**, **GitHub Copilot**, and others while editing component logic
- Improve productivity and consistency when iterating on complex components

If you’re working with custom Langflow components, I’d love to hear your feedback.
